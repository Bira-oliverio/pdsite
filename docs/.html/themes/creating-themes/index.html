Creating a new theme

Apdsite theme can be as minimal as a folder containing a single template.html file that provides a pandoc template for rendering HTML pages. Arbitrary additional files (CSS, JS, 404 pages, etc) can be provided as well -- their contents and folder structure will just be copied verbatim into the site output directory. The theme name is given by the name of the container folder once it has been copied into the themes subdirectory.
